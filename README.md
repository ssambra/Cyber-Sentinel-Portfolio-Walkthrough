# Operation Sentinel – LinkedIn Hunter

Operation Sentinel LinkedIn Hunter parses security exports to detect anomalous logins against a trusted baseline, mapping directly to DORA Article 10 incident detection requirements. Built in Python for rapid audit evidence generation, it reduces manual IAM log review from two hours to under 30 minutes. Designed for EU fintech and banking environments requiring NIS2-compliant identity monitoring.

## Run the demo
python3 linkedin_hunter.py
